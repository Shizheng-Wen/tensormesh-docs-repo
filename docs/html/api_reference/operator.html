Condenser

class Condenser(dirichlet_mask: Tensor, dirichlet_value: Optional[Tensor] = None)[source]

Bases: object

Static Condensing Operator for Dirichlet Boundary Condition

\[K_{inner} u_{inner} = f_{inner} - K_{ou2in} u_{ou2in}\]

Examples

import torch
from tensormesh.operator import Condenser
from tensormesh.sparse import SparseMatrix

# Create a simple system with 4 DOFs
n_dof = 4
dirichlet_mask = torch.tensor([True, False, False, True]) # Fix DOFs 0 and 3
dirichlet_value = torch.tensor([1.0, 2.0]) # Set values at fixed DOFs

# Create condenser operator
condenser = Condenser(dirichlet_mask, dirichlet_value)

# Create a sample stiffness matrix
row = torch.tensor([0,0,1,1,2,2,3,3])
col = torch.tensor([0,1,1,2,2,3,2,3])
val = torch.ones(8)
K = SparseMatrix((val, (row, col)), shape=(n_dof, n_dof))

# Create RHS vector
f = torch.ones(n_dof)

# Condense system
K_inner = condenser.condense_matrix(K) # Get condensed stiffness matrix
f_inner = condenser.condense_rhs(K, f) # Get condensed RHS vector

# Solve condensed system
u_inner = torch.linalg.solve(K_inner.to_dense(), f_inner)

# Recover full solution
u = condenser(K, u_inner)
Parameters:

  • dirichlet_value (torch.Tensor) – 1D tensor of shape \([n_{\text{dof}}]\) or \([n_{\text{outer_dof}}]\) the value of the dirichlet boundary condition. The default value is 0.
